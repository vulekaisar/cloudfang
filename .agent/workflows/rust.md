---
description: Recommended workflows for Rust development
---

# Build
Run `cargo build` to compile the project.
// turbo
cargo build

# Test
Run `cargo test` to execute unit and integration tests.
// turbo
cargo test

# Check
Run `cargo check` to quickly verify that the code compiles (faster than build).
// turbo
cargo check

# Lint
Run `cargo clippy` to catch common mistakes and improve code quality.
// turbo
cargo clippy

# Format
Run `cargo fmt` to automatically format the code.
// turbo
cargo fmt
